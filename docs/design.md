# CATI 设计规范

## 整体风格
- **关键词**：简洁、温暖、可爱、现代
- **参考**：MBTI 小人图标风格（简洁几何人物）+ 低多边形（low-poly）美学
- **氛围**：轻松愉悦，像在咖啡馆做测试

## 猫咪插图设计规范（MBTI 小人风格）

- 全身像：每只猫包含完整的头部、身体、四肢、尾巴
- 圆润简洁：使用圆形、椭圆、圆角矩形、简单贝塞尔曲线，无尖锐三角形
- 扁平卡通：纯色填充，少阴影，类似 16personalities 的小人图标风格
- 每只猫姿势/表情不同以体现性格：
  - 缅因猫：端庄坐姿，高大挺拔
  - 布偶猫：优雅坐姿，毛茸茸温柔
  - 狸花猫：站立警觉，矫健敏捷
  - 暹罗猫：优雅坐姿，大耳重点色
  - 英短：圆胖坐姿，憨态可掬
  - 三花猫：坐姿，三色斑块分布
  - 美短：警惕坐姿，运动健壮
  - 橘猫：侧躺放松，圆胖可爱
- 统一 200×200 viewBox，首页缩略图约 60×60，结果页约 160×160
- 腮红点缀增加可爱感
- 部分猫加入品种特征（条纹、重点色、斑块等）

## 动画

- 屏幕切换：淡入淡出（opacity transition, 300ms）
- 按钮 hover：轻微放大 + 颜色加深（transform scale 1.03, 200ms）
- 进度条填充：平滑过渡（width transition, 400ms）
- 结果页猫图：轻微弹入（scale 0.8→1.0, 500ms ease-out）
